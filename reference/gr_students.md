# Dummy student data for the Graduation Rates scripts

Dummy student data for the Graduation Rates scripts

## Usage

``` r
gr_students
```

A data frame with 101 rows and 13 columns

one student (a program-switcher) has a 4th category \[1\] for easy
tracking
